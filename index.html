HTML (HyperText Markup Language)

O HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar e organizar o conteúdo das páginas web. É fundamental para o desenvolvimento de sites e aplicações na web, pois permite definir a estrutura de um documento, a colocação de textos, imagens, links, tabelas, formulários e outros.
Esta linguagem é interpretada pelo browser, o que significa que podemos vizualizar o resultado do código, abrindo apenas o ficheiro num navegador da nossa preferência.

Estrutura do código 


<!DOCTYPE html>
    <html lang="pt">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título da Página</title>
    </head>

    <body>
    <!-- Conteúdo visível da página -->
        <h1>Bem-vindo ao meu site!</h1>
        <p>Este é um parágrafo de exemplo.</p>
    </body>
</html>

CSS - Cascade Style Sheets

O CSS (Cascading Style Sheets) é uma linguagem utilizada para definir a apresentação de documentos HTML.
Enquanto o HTML define a estrutura e o conteúdo de uma página web, o CSS define como esses elementos serão exibidos, incluindo cores, fontes, tamanhos, posicionamento e animações.

Exemplo CSS Inline (como atributo da tag HTML)


<p style="color: red;">Este texto está vermelho.</p>

Exemplo CSS Interno (dentro do àrea HEAD da página)


<!--Altera para verde, a cor do texto dos elementos h1-->

<head>
    <style>
        h1 {
            color: green;
        }
    </style>
</head>

Exemplo CSS externo (escrito num ficheiro .css externo)


html

<link rel="stylesheet" href="styles.css">  

ficheiro .css

body {
        background-color: #f0f0f0;
        font-family: Arial, sans-serif;
}

JavaScript

O JavaScript é uma linguagem de programação essencial para desenvolvimento web. É usada para tornar páginas da web interativas, manipular elementos HTML e CSS. Pode tmabém, ser utilizada para desenvolver aplicações complexas do lado do servidor usando Node.js.

Exemplo de uma estrutura condicional


<script>
let hora = 15;

if (hora < 12) {
    console.log("Bom dia!");
    } else if (hora < 18) {
        console.log("Boa tarde!");
    } else {
         console.log("Boa noite!");
    }    
</script>
